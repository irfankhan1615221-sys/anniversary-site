# Anniversary Site

Upload these files to a public GitHub repository and enable GitHub Pages (Settings → Pages → Deploy from a branch → `main` /root). Your site will be available at:

```
https://<your-username>.github.io/<repository-name>/
```

Add your images with the exact filenames in `assets/images/`:
- IMG_20250517_011051_540.jpg
- IMG_20250714_091914_701.jpg
- 20250907_143845.jpg

(Optional) Put a song at `assets/audio/song.mp3` and uncomment the `<audio>` tag in `index.html`.
